Survive Sim

Game Programmer | Gameplay Engineer | Level Designer

Game Engine: Unreal

Project Overview

A 3D first-person puzzle/survival game currently in development. Players must solve puzzles using their 2D character companion, which they control inside a computer. The gameplay is about switching between solving puzzles with the 2D character and making sure that the player is safe by checking for nearby monsters or threats.

Key Features I Worked On

  • Implemented the switching of characters and display the point of view in a monitor.
