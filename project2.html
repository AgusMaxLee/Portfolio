Ruins of Prospero

Game Programmer | Gameplay Engineer | Game Designer

Game Engine: Unity

Project Overview

A 3rd person RPG set in a low poly world, where players control a wizard capable of switching between water, fire, earth elements, and a sword-only mode. Each element provides unique abilities and passive effects, changing the player's HP, mana, and damage. The theme of the game while developing it was "side effects", therefore drawing attention to the strategic element-switching mechanic. Players must defeat three main elemental bosses and defeat small ghosts in the open-world map. Additionally, they can collect apples as a healing option.

Key Features I Worked On

  • Developed the character movement and all of it's animations.
  • Implemented a system allowing the wizard to switch between water, fire, earth, and sword-only modes and having different abilities for each one.
  • Developed mechanics where switching between elements affects the player's HP, mana, and damage.
  • Improved the AI logic of the three main bosses representing water, earth, and fire elements.
  • Improved the open-world map and populated it with small ghost enemies that attack the player.
  • Implemented all the UI elements of the game.
  • Integrated a system for players to use apples to restore health during gameplay.
