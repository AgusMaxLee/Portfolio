Team LATAM

Game Programmer — Gameplay Engineer

Unity 2D Platformer Shooter Team Project

Project Overview

This was a team project and the whole idea was to make something frustrating on purpose. You're playing a 2D shooter trying to get through levels, but an AI named Jeff keeps sending you pop-up messages right in the middle of fights. There are no checkpoints either, so every time you die you start from the beginning. It sounds annoying and it is, but once you get into it it's actually really fun.

What I worked on

  • Built all the player mechanics from scratch, movement, jumping and shooting.
  • Got all the character animations feeling smooth with Unity's Animator.
  • Programmed the whole Jeff pop-up dialogue system.
  • Made the first set of enemy types with their patrol and attack behaviors.

Tech Stack

Unity C# Animator Physics2D
