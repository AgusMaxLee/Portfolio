Team LATAM

Game Programmer — Gameplay Engineer

Unity 2D Platformer Shooter Team Project

Project Overview

A 2D shooter platformer where the player must eliminate enemies and reach the end of each level while constantly being interrupted by pop-up messages from an AI named Jeff. The game is deliberately difficult — deaths are frequent and there are no checkpoints — making it a tense, frustration-fueled experience with a comedic twist. Smooth character animations, a variety of enemy types, and an escalating difficulty keep the player on edge throughout.

My Contributions

  • Developed the player's movement, jumping, and shooting mechanics from the ground up.
  • Implemented smooth transitions between all character animations using Unity's Animator.
  • Programmed the AI-driven pop-up dialogue system for the in-game character Jeff.
  • Built and balanced the first set of enemy types, including their patrol and attack behaviors.

Tech Stack

Unity C# Animator Physics2D
