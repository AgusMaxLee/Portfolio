Neon Runner

Programmer & Level Designer

Unreal Engine 5 3rd Person Runner Team Project

Project Overview

A fast-paced third-person speedrunning game set in a futuristic world. The objective is simple: get from point A to point B as fast as possible while collecting points along the way. What sets it apart is a color-switching mechanic — players can toggle between blue and red modes, each of which changes the layout and obstacles of the map in real time, adding a layer of strategy to pure speed.

My Contributions

  • Designed levels that challenge players and reward speedrunning strategies.
  • Developed the death and checkpoint system to save and restore player progress.
  • Assisted in implementing the blue/red color-switching mechanic that dynamically alters the map.
  • Contributed to building and debugging the game's UI elements.
