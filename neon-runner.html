Neon Runner

Programmer & Level Designer

Unreal Engine 5 3rd Person Runner Team Project

Project Overview

A third-person speedrunner made in Unreal Engine 5. The goal is to get from A to B as fast as you can while picking up points. What makes it interesting is that you can switch between blue and red mode at any time and it actually changes the level layout on the fly. So you have to think about when to switch and it adds a whole extra layer on top of just running fast.

What I worked on

  • Designed the levels to make speedrunning feel satisfying and rewarding.
  • Built the death and checkpoint system so players don't lose all their progress.
  • Helped build the color switching mechanic that changes the map in real time.
  • Worked on the UI and fixed a bunch of bugs in it.
