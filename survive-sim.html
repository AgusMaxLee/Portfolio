Survive Sim

Gameplay Programmer & Level Designer

Unreal Engine 5 FP Puzzle / Survival In Progress

Project Overview

This one is still in development but the concept is something I'm really excited about. You're in a 3D first person world but you also have a 2D character that lives inside a computer screen and you can switch between them. The whole game is about managing both at the same time. You need the 2D character to solve puzzles but you can't ignore the 3D world either because monsters can get you while you're not paying attention.

It creates this constant tension where you're never fully safe no matter which perspective you're in. That's what I love about the concept.

What I worked on

  • Built the character switching system between the 3D world and the 2D screen.
  • Made the in-game monitor that actually renders the 2D world inside the 3D environment.
