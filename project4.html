Neon Runner

Game Programmer | Gameplay Engineer | Level Designer

Game Engine: Unreal Engine 5

Project Overview

A 3rd person futuristic runner where the objective is to go from point A to point B as quickly as possible while collecting points. It's a speedrunning game with smooth, fast-paced animations and mechanics. Players can switch between blue and red modes, changing the map dynamically based on the current color.

Key Features I Worked On

  • Designed maps that challenge players and improve the speedrunning experience.
  • Developed a system to handle player deaths and incorporated checkpoints to save progress.
  • Assisted in creating the mechanic that allows players to switch between blue and red modes.
  • Contributed to the development and fixing of the game's user interface.
