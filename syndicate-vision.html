Syndicate.Vision

3D Web Environment — Multiplayer Developer & Environment Artist

Three.js JavaScript 3D Web Environment Client Project

Project Overview

A fully interactive 3D online store built for Syndicate.Vision, a real client. Working in a small team, we built a web-based virtual shopping environment where visitors can walk around the store, browse products, and — the part that makes it special — see other shoppers moving around in real time. Think of it as a living, breathing store rather than a static webpage: you're never alone in it.

The environment was modeled in Blender and brought to life in the browser using Three.js. Real-time presence was powered by a Firebase Realtime Database, syncing each user's position and movement across all connected sessions.

My Contributions

  • Built and refined sections of the 3D store environment in Blender, focusing on layout, atmosphere, and visual detail.
  • Implemented performance optimizations to keep the Three.js scene smooth in-browser — geometry merging, texture atlasing, draw call reduction.
  • Designed and developed the in-world UI elements visible within the 3D space.
  • Architected and implemented the full multiplayer system using Firebase Realtime Database — live syncing of player positions, spawning/despawning remote avatars, and handling connection state.

The multiplayer challenge: making the experience feel alive without a dedicated game server — just Firebase, JavaScript, and Three.js. Every visitor's position is written to the database and read by everyone else, updating avatars in real time at 60fps.
